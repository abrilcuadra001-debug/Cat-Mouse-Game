# Cat Mouse Game 🐱🐭

A Java Swing game where a cat chases a mouse through a maze.

## Features
- Maze-style movement
- Cheese power-ups
- Unlockable cat skins
- Saved progress
- Sound effects with volume control

## Controls
- WASD: Move
- R: New round
- 1–4: Change skins
- M: Mute
- + / -: Volume

## How to Run
javac CatMouseGame.java  
java CatMouseGame
